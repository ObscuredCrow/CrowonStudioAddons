Monster Waypoints

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: Yes Tutorial Video: No

Description

Allows Monsters to patrol any number of waypoints.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script Monster.cs. (Click to Expand)
    Locate this line: 
    
if (EventMoveRandomly())
{
	// walk to a random position in movement radius (from 'start')
	// note: circle y is 0 because we add it to start.y
	Vector2 circle2D = Random.insideUnitCircle * moveDistance;
	Navigate(startPosition + new Vector3(circle2D.x, 0, circle2D.y), 0);
	return "MOVING";
}

    Modify to match:

    
if (EventMoveRandomly())
{
	
	// addon system hooks
	Utils.InvokeMany(typeof(Monster), this, "OnEventMoveRandomly_");

	return "MOVING";
}
