Npc Restrictions

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSNPCRESTRICTIONS
Core Changes: Yes Tutorial Video: No

Description

A simple AddOn that allows you to set various restrictions to NPCs. Only players who fulfill all requirements can interact with the Npc (limited to Guild, minLevel, required Skill + Skill Level, Quest completion, limited to Class etc.)
