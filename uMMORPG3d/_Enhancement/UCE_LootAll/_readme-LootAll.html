Loot All

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: Yes Tutorial Video: No

Description

Adds the ability to loot all items from a corpse by clicking a button.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script UILoot.cs. (Click to Expand)
    Locate this line: 
    
	slot.amountOverlay.SetActive(items[i].amount > 1);
	slot.amountText.text = items[i].amount.ToString();
}

    Modify to match:

    
	slot.amountOverlay.SetActive(items[i].amount > 1);
	slot.amountText.text = items[i].amount.ToString();
}
GetComponent<UCE_LootAll>().LootAll();

Installation

  1. [Required] Expand your Loot Window then right click on LootPanel then under UI add a Button.
  2. [Required] Add all Scripts from Scripts [Add to Loot] to your Loot Window right below UILoot.
  3. [Required] Drag and drop your newly created button onto Loot All Button under that script we just added.
