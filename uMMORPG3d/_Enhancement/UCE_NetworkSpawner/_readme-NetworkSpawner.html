Network Spawner

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

A very simple network spawner to fix the "scene id not found" issue. It spawns prefabs after the scene is loaded.

Usage

  1. [Required] Place UCE_NetworkSpawner prefab in your Scene.
  2. [Required] Edit the list on the NetworkSpawner and add all objects and their locations where you want them to spawn.
  3. [Required] Make sure those objects have a NetworkIdentity and are listed in your NetworkManagers spawnable prefabs.

Video

Extras

  • Note that those objects can also be other types of spawners or trigger zones by themselves.
  • After spawning, the spawner destroys itself.
