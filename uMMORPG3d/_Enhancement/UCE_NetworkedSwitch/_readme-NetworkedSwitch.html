Networked Switch

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

Allows any player within range to activate a switch per click. This switch will toggle activations/deactivations of a linked game object. The original plan was to use this as a "light switch" and a "networked lamp", but i can imagine a few other uses as well.

Usage

  1. [Required] Add all Prefabs from Prefabs [Drag to Scene] to your Scene.
  2. [Required] Select the NetworkedSwitch game object and add to the Activated Objects list the ActivateableObject game object.

Video

Extras

  • In order to test, access the NetworkedSwitch with a player, this will activate/deactivate the ActivateableObject.
