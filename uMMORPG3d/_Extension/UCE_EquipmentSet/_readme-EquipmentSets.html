Equipment Sets

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSEQUIPMENTSETS
Core Changes: No Tutorial Video: No

Description

Allows to add “Individual, Partial and Complete Set Bonus” to equipment items that kicks-in when the player has one, some or all items of a set equipped. Individual, Partial and Complete Set Bonus can be active at the same time. Each Set Bonus can alter all attributes and secondary stats when using in conjunction with "UCE Attributes" AddOn.

Extras

  • Use the provided example equipment or simply create two or more Equipment Items via the context menu and edit their settings.
  • A equipment set consists of two parts: The actual equipment items and a "Equipment Set" template. You have to create at least two equipment items and exactly one equipment set template for each equipment set.
  • Edit your items as usual and assign the "Equipment Set" template to each item that belongs to the same set as well.
  • There are three different kinds of "set bonus" in total, each equipment set can feature one, some or all of them.
  • Individual Set Bonus - You edit this bonus on each item itself, so it can be unique on a per-item basis. It's activated on each item and only when the full set is worn.
  • Partial Set Bonus - This is defined on the "Equipment Set Template" and is activated when a certain amount of items is equipped. You can define the number of set items that have to be equipped (like 4 out of 6 etc.).
  • Complete Set Bonus - This final bonus is also defined on the "Equipment Set Template" and is activated only when all set-items are equipped (e.g. the full set).
