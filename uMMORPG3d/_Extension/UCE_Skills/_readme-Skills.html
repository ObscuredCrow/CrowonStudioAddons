Skills

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSMELEE, _CSPROJECTILE
Core Changes: No Tutorial Video: No

Description

Contains a large array of new skills for your game that will greatly enhance gameplay.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Create a new Skill via the Context Menu.
  2. [Required] Create a new Animator State name it the same as your Skill.
  3. [Required] Add a Animation Parameter as a bool and name it the same as your Skill.
  4. [Required] Add your Transitions, look at other skills for an example.

Screenshots

None.

Video

None.

Extras

  • When using a muzzle or impact skill effect, you also have to add a "OneTimeTargetSkillEffect" component to it (like in default uMMORPG).
  • All important settings are on the Skill itself, you only edit those. Only edit the projectile prefabs if you want to change how they look.
  • Target Melee Skill - functions like a melee skill in uMMORPG with extra options but no physics. Low Network Traffic & Calculations.
  • Targetless Melee Skill - a physics based melee attack using a collider. Allows for targetless hack+slash attacks. Medium-High Network Traffic & Calculations.
  • ArrowProjectile - functions like a ranged skill in uMMORPG with extra options but no physics. Destroyed on Impact. Low Network Traffic & Calculations.
  • ColliderProjectile - a physics based projectile with a collider. Allows for penetration projectiles. Destroyed after travelled distance/lifetime. Medium-High Network Traffic & Calculations.
  • BallisticProjectile - A physics based projectile with a collider. Allows for penetration projectiles & arced fly-path. Destroyed after travelled distance/lifetime. High Network Traffic & Calculations.
