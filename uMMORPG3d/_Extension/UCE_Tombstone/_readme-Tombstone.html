Tombstone

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

Leaves a tombstone behind when a entity dies.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Add all Prefabs from Prefabs [Add to Entity] to your Entities.
  2. [Required] Make sure the Tombstone object is added to your Network Managers spawnable prefabs list.

Screenshots

None.

Video

None.

Extras

  • You can use any kind of game object as Tombstone, the default one has all required components on it already.
  • You can define more than one tombstone, each one that is rolled for will be spawned.
