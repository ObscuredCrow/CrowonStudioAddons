Item Rarity

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSITEMRARITY
Core Changes: Yes Tutorial Video: No

Description

Item rarity adds flavor effects to your in-game items by bring a outline to your items based on the quality level you set. It currently has the types: Poor (grey), Common (white), Uncommon (green), Rare (blue), Epic (magenta), and Legendary (yellow). It also adds the color effect to the name of the item in the items tooltip. The following menus all display this flavor: Equipment, Inventory, Loot, Item Mall, Player Trading, Npc Trading, and Crafting.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script UIItemMall.cs. (Click to Expand)
    Locate this line: 
    int currentCategory = 0;

    Modify to match:

    public int currentCategory = 0;

Installation

  1. [Required] Add the Script UCE_ItemRarity from Scripts [Add to Windows] to all of the following windows: Equipment, Inventory, Loot, PlayerTrading, NpcTrading, ItemMall, and Crafting.
  2. [Required] Set the Slot Type on all of the following windows based on the window, under UCE_ItemRarity: Equipment, Inventory, Loot, PlayerTrading, NpcTrading, ItemMall, and Crafting.
  3. [Required] Add the script UCE_RaritySlot to all of the following slot prefabs: Equipment, Inventory, Loot, PlayerTrading, NpcTrading, ItemMall, and Crafting.
  4. [Required] Add the SquareBorder to all of the following slot prefabs, as a image UI above all children: Equipment, Inventory, Loot, PlayerTrading, NpcTrading, ItemMall, and Crafting.
  5. [Required] Link the SquareBorder to the Outline on each slots UCE_RaritySlot script.
