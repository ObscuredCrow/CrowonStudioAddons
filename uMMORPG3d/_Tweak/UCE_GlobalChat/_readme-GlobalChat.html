Global Chat

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

This AddOn will modify the default chat and add a global channel to it. Posting on that channel does come with certain limitations like a cooldown time, a (optional) required item, a minimum level and so on.

Installation

  1. [Required] Attach all Prefabs in Prefabs [Attach to PlayerChat] to your Player Prefabs PlayerChat.

Video

Extras

  • Has options for requiring an item before you can chat (leave the item field blank for no requirement), minimum player level before global chat, and also has rate limiting by setting how many seconds between global chat messages. Each limiting factor also has custom error messages set in the inspector that will be shown in the player’s chat on the “Info” channel (red messages). Note that the item just needs to be in the player’s inventory. It is not something they need to use to be able to chat (that would require a player variable to track whether they have used the item).
