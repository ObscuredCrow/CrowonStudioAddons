Loot Rules

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSLOOTRULES
Core Changes: Yes Tutorial Video: No

Description

Allows you to set monsters to improve the looting situation. Right now everyone can loot. When using this AddOn, only a specific group of players can loot.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

Extras

  • The player who defeated the monster, can always loot it.
  • Lift Rules after - Set this to a number of seconds, after that all loot rules are removed (disabled when set to 0).
  • If you want Monsters to be "Realm Lootable" you must also install "UCE PVP Zones" AddOn.
