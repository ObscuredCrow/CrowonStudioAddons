Walking Npc

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: Yes Tutorial Video: No

Description

Allows NPCs to walk around randomly like Monsters or to patrol any number of waypoints. Npc's also suspend their next movement while a player is near them.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script Npc.cs. (Click to Expand)
    Locate this line: 
    [Server] protected override string UpdateServer() { return state; }

    Modify to match:

    //[Server] protected override string UpdateServer() { return state; }

Installation

  1. [Required] Add new Animation Parameter(s) to your NPCs controller: MOVING.
  2. [Required] Add new Animation to your NPCs controller for MOVING and transition to it.

Extras

  • Waypoints are simple transforms and could be any gameobject (like grass on the ground) you can also create "empty" game objects and use those as waypoints.
  • You can mix both Random Movement and Waypoint Movement and set a ratio for the Npc to switch between both types. This way you can have Npcs that move a little bit at random, then continue to the next waypoint and so on.
