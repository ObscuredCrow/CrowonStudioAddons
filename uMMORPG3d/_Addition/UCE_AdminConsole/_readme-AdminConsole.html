Admin Console

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

A chat command based admin console. Its not a beauty, but its very powerful: Customizeable, Scriptable, Extendable Commands. Nominate/Demote Admins on the fly, add Admin levels, admin level based commands, commands that target individual players or parties, guilds. Ships with over a dozen admin commands.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Move all Content from Resources [Move to Resources] to your Resources folder.
  2. [Required] Add all Scripts from Scripts [Attach to Players] to all Player Prefabs.
  3. [Required] In the new component, open and edit Commands on each one, add enough elements to match the amount of commands and assign each command.
  4. [Required] Then edit Target Tags to each have a key assigned.

Extras

  • The very first account you create, will always be your Master Admin account. This one has the highest admin level and is able to upgrade or downgrade other players to admins.
  • Almost all actions the admins can do, target only ONLINE PLAYERS, it is not possible yet to add items to offline players inventory for example!
  • There is a hidden problem in the uMMORPG3d core asset that prevents characters from being deleted. It overwrites the deleted entry all the time and there is no easy fix for this yet. It is located in Database.cs file in CharacterSave function.
