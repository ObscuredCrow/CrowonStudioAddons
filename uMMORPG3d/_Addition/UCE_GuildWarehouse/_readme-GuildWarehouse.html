Guild Warehouse

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSGUILDWAREHOUSE
Core Changes: Yes Tutorial Video: No

Description

this addon adds a "Guild based warehouse" to the game. You must be in a Guild to access and only your guild members can deposit and withdraw items and gold to the warehouse. This allows players to share items across their guild members. You can use the original warehouse script in addition to this one, if you want.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script Player.cs. (Click to Expand)
    Locate this line: 
    
[Command]
public void CmdLeaveGuild()
{
	// validate
	if (InGuild())
	GuildSystem.LeaveGuild(guild.name, name);
}

    Modify to match:

    
[Command]
public void CmdLeaveGuild()
{
	// validate
	if (InGuild()) 
	{
		GuildSystem.LeaveGuild(guild.name, name);
		resetGuildWarehouse();
	}
}

Installation

  1. [Required] Add all Prefabs from Prefabs [Drag to Canvas] to your Canvas.
  2. [Required] Add all Scripts from Scripts [Attach to NpcDialogue] to your NpcDialogue.
  3. [Required] Move all Content from Resources [Move to Resources] to your Resources folder.
  4. [Required] Link ButtonGuildWarehouse and UCE_UI_GuildWarehouse to your NpcDialogue.
  5. [Required] Link Inventory Gold Text to your GuildWarehouse.
  6. [Required] Link Warehouse Config to all Player Prefabs.
