Traits

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSTRAITS
Core Changes: No Tutorial Video: No

Description

Required: Attributes and Character Creation

Adds customizeable character traits to players (right now, all player prefabs are the same). Traits can be assigned during character creation. Traits can modifiy almost everything from attributes to stats, skills, resistances etc.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Move all Content from Resources [Move to Resources] to your Resources folder.

Screenshots

None.

Video

None.

Extras

  • Edit the new "UCE Traits" section on your player prefabs and assign a number of trait points to them.
  • Traits are only available during character creation and cannot be altered afterwards anymore (at least not yet).
  • Traits allow your players to modify almost every aspect on their characters (stats that is, not visual aspects).
  • Carefully edit the number of trait points each player prefab has available, as well as the trait cost of each trait.
  • Assign a unique group ID to traits, to limit your players to just one trait of that group.
  • Check the "Allowed Classes" section on the Trait template, to limit a trait to a certain class.
