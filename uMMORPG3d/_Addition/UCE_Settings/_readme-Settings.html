Settings

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSSETTINGS
Core Changes: Yes Tutorial Video: No

Description

Adds a settings menu to uMMORPG that will greatly enhance player control: Keybindings, Gameplay, Video, and Sound.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script UIChat.cs. (Click to Expand)
    Locate this line: 
    if (player)

    Modify to match:

    if (player && settingsVariables.isShowChat)
  2. [Required] Open the following script Player.cs. (Click to Expand)
    Locate this line: 
    
float horizontal = Input.GetAxis("Horizontal");
float vertical = Input.GetAxis("Vertical");

    Modify to match:

    
//float horizontal = Input.GetAxis("Horizontal");
//float vertical = Input.GetAxis("Vertical");
  3. [Required] Open the following script Player.cs. (Click to Expand)
    Locate this line: 
    
return entity != null && entity is Player && entity != this &&
	CanStartTrade() && ((Player)entity).CanStartTrade() &&
	Utils.ClosestDistance(collider, entity.collider) <= interactionRange;

    Modify to match:

    
return entity != null && entity is Player && entity != this &&
	CanStartTrade() && ((Player)entity).CanStartTrade() &&
	Utils.ClosestDistance(collider, entity.collider) <= interactionRange &&
	!((Player)entity).isBlockingTrade;
  4. [Required] Open the following script Player.cs. (Click to Expand)
    Locate this line: 
    if ((!InParty() || !party.IsFull()) && !other.InParty())

    Modify to match:

    if ((!InParty() || !party.IsFull()) && !other.InParty() && !other.isBlockingParty)

Installation

  1. [Required] [With MainMenu] Add all Prefabs from Prefabs [Drag to UCE_UI_MainMenu] to your UCE_UI_MainMenu.
  2. [Required] [Without MainMenu] Add all Prefabs from Prefabs [Drag to Canvas - wo MainMenu] to your Canvas.
  3. [Required] Replace all FaceCamera Scripts with Scripts [Replace all FaceCameras].

Extras
