Guild Upgrades

Version: 2021.100 Requirements: uMMORPG3d Script Define: _CSGUILDUPGRADES
Core Changes: Yes Tutorial Video: No

Description

Requires: Guild Warehouse

Allows the guild master or vice to upgrade the guild. Upgrading a guild increases member capacity and can reward the upgrading player with an item. The guild level can be used in all "UCE Interaction Requirements" to control access to teleporters etc. as well.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. [Required] Open the following script UITarget.cs. (Click to Expand)
    Locate this line: 
    player.guild.CanInvite(player.name, target.name) &&

    Modify to match:

    player.UCE_CanInvite(player.name, target.name) &&

Installation

  1. [Required] Add all Prefabs from Prefabs [Drag to Canvas] to your Canvas.
  2. [Required] Add all Scripts from Scripts [Attach to NpcDialogue] to your NpcDialogue.
  3. [Required] Move all Content from Resources [Move to Resources] to your Resources folder.
  4. [Required] Link ButtonGuildUpgrade to your NpcDialogue.
  5. [Required] Link UCE Guild Upgrades to all Player Prefabs.
