Mail

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

This AddOn adds a full mail message client to the game. Players can send mail between each other, the mail is delivered to a player even while offline and can be read later. Improves communication quite a bit.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Move all Resources from Resources [Move to Resources] to your Resources.
  2. [Required] Link UCE Mail Settings to your NetworkManager.
  3. [Required] Link UCE Mail Settings to your Player Prefabs.
  4. [Required] Add all Prefabs from Prefabs [Drag to Canvas] to your Canvas.
  5. [Required] Add all Prefabs from Prefabs [Drag to PanelButtons] to your NpcDialogue PanelButtons.
  6. [Required] Attach all Scripts from Scripts [Attach to NpcDialogue] to your NpcDialogue.
  7. [Optional] Add all Prefabs from Prefabs [Optional Drag to Shortcuts] to your Shortcuts.
  8. [Optional] Attach all Scripts from Scripts [Optional Attach to Shortcuts] to your Shortcuts.
