Main Menu

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: Yes Tutorial Video: No

Description

Adds a main menu similar to most games when you hit escape. It will give you the options of resume, change character, logout, and exit. If you're using UCE_Options it will also give you a settings option as well.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. Requires a specific core changes, see below!

Installation

  1. [Required] Add all Prefabs from Prefab [Add to Canvas] to your Canvas.
  2. [Required] Add all Prefabs from Prefab [Add to Shortcuts] to your Shortcut Buttons.
  3. [Required] Add all Scripts from Scripts [Add to Shortcuts] to your Shortcut GameObject.
  4. [Required] Attach all needed components onto UCE_UI_MainMenuShortcuts in your Canvas, under Shortcuts.
  5. [Required: Core Change] Open the following NetworkManagerMMO. (Click to Expand)
    Locate this line: 
    uiPopup.Show("Disconnected.");

    Add this Above It:

    if (!changingCharacters)
  6. [Required: Core Change] Open the following UILogin. (Click to Expand)
    Locate this line: 
    if (manager.state == NetworkState.Offline || manager.state == NetworkState.Handshake)

    Add this Above It:

    if (!manager.changingCharacters)
  7. [Optional] Attach any game window panel you want hidden when UCE Main Menu is opened to UCE_UI_MainMenu script under Close Windows.
  8. [Optional] Set the combat timer for how long before a player can logout after combat.
  9. [Optional] Set the logout timer for how long it takes before a player logs out.
