Npc Healer

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

Allows healing at a Npc (Inn/Healer) with variable or fixed prices. Can remove buffs as well.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Add all Prefabs from Prefabs [Drag to Canvas] to your Canvas.
  2. [Required] Add all Prefabs from Prefabs [Drag to NpcDialogue] to your NpcDialogue.
  3. [Required] Add all Scripts from Scripts [Attach to NpcDialogue] to NpcDialogue.
  4. [Required] Link ButtonHealer to your NpcDialogue.

Extras

  • Gold Cost - the fixed cost to access this healer, is used as a fixed price when "Scale Cost" is unchecked, otherwise its used as multiplicator.
  • Scale Cost - scales the gold cost according to the amount of health/mana/buffs the player requires to be healed. This can result in a very high cost very quickly.
  • Scale cost is calculated by multiplying the "Gold Cost" by the amount of health/mana/buffs that needs to be healed/removed.
  • Unequip Max Cursed Level - does not work with Scaled cost, use fix cost instead.
  • To prevent confusion, it is recommended to create one healer per service offered and not to mix all of them into one Npcs.
