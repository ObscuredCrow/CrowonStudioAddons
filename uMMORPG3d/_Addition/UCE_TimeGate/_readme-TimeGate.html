Time Gate

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

This Timegate allows you to add teleporters with opening times to your game. These teleporters act as entrances to dungeons, shops, festival areas or special terrains that are only accessible during special times of the year or within certain limitations.

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] Add all Prefabs from Prefabs [Add to Canvas] to your Canvas.
  2. [Required] Add all Prefabs from Prefabs [Add to Scene] to your Scene.

Extras

  • Set a (optional) number of maximum visits per player. This number is increased by one, each time a player enters the teleporter. Once the maximum is reached, the player cannot access the teleporter anymore until it is reset. A teleporter is reset as soon as it is closed. This only works in conjunction with opening times.
  • Set a (optional) amount of hours between visits. This allows you to pace how quickly a player can enter (or re-enter) the teleporter in sequence. So you can have a one hour delay between each use.
  • Set a start day and a end day, the teleporter will only be accessible during these days. You can combine this option with the active month option.
  • Set a active month, the teleporter will only be accessible during this month. You can combine this option with the start and end day option.
