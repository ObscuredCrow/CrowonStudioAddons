Chairs

Version: 2021.100 Requirements: uMMORPG3d Script Define: No
Core Changes: No Tutorial Video: No

Description

This AddOn allows you to setup chairs to be occupied by 1 Player at a time. You can setup a Parent GameObject with Children that contain the UCE_ChairAddon to allow multiple seats per chair (Children each require Colliders, Spacing them apart for easy accessibility).

Core Changes

Some add-ons require changes to the core framework in order to function. This add-on requires the following change(s):

  1. None.

Installation

  1. [Required] For all Player Prefab Animators, Add a Bool labeled "Sit", Add a new Empty State, Add a Transition from 'Any State' to your new Empty State, Select the Transition and Add a new Condition of Bool 'Sit' to True. Click Settings in the new State and De-Select Can Transition to Self.
  2. [Required] Also add a new condition to the Idle state of 'Sit' to false.
  3. [Required] To setup a Chair, follow the same steps when creating a base UCE_InteractableObject prefab, but instead replacing script 'UCE_InteractableObject' with 'UCE_ChairAddon' Thats it!
